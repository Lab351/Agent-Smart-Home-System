# 智能家居 Agent 系统测试场景用例

## 测试环境

### 硬件环境
- Jetson Orin (ARM64) x3 - Room Agent
- ESP32 x3 (客厅、卧室、书房) - BLE Beacon
- 智能设备：灯光、窗帘、空调等

### 软件环境
- Python 3.12
- MQTT Broker: 120.78.228.69:1884 (或本地 Broker)
- 本地网络: 192.168.1.x

### Agent 组件
- Personal Agent: 手机/手表应用
- Room Agent: 边缘设备上运行
- Central Agent: NAS/云端运行
